# tlibpng
A simple png decoder for practice. 

It would receive a png file path through shell parameter, decode it to a RGBA array, and then display it via [SDL2 lib](https://www.libsdl.org/). 

## Usage

```
mkdir build
cd build
cmake ..
make
cd bin
./png_decoder PNGFILEPATH
```
